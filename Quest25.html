ALIGN O comando align serve para alinhar textos em determinadas áreas da tela do computador.
VALING O comando Valign tem a função de fazer o alinhamento dos textos verticalmente.
CELLSPACING O comando define o espaço entre células através de pixels.
CELLPADDING O comando define o espaço entre células e a parede da tabela através de pixels.
ROWSPAN O comando define o número de rows de uma célula.
COLSPAN O comando define o número de colunas que deve aparecer.
WIDTH Define a largura da tabela.
